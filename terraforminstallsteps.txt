
Installation Steps:

Download the latest Terraform release ZIP for Windows from the official site.
Unzip it and place the terraform.exe somewhere like C:\Terraform.
Add this folder to your System PATH (Control Panel > System > Advanced system settings > Environment Variables).
Verifying Terraform Installation:
After installation, open a new terminal or command prompt window and type:
terraform version
